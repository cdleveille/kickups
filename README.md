## Local Setup

1. Install [Bun](https://bun.sh)
2. Install package dependencies: `bun i`
3. Install [Docker Desktop](https://www.docker.com/products/docker-desktop)
4. Initialize database container: `bun db:up`
5. Start app: `bun dev`
